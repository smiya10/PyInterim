Useful Modules

  • Python Standard Library
    • math - common math functions
    • string - string manipulation
    • datetime - dates and times
    • time - time access and conversions (similar to datetime)
    • os - portable operating system-dependent interfaces
    • sys - system-specific stuff
    • pickle - saving Python data
    • csv - reading and writing CSV (Comma Separated Values) files
    • re - regular expression matching
    • collections - more complex container types
    • random - pseudo-random numbers
    • argparse - command line argument parsing
    • curses - terminal character display
    • tkinter - default Python graphics
  • Other Libraries
    • numpy - fast array manipulation
    • matplotlib - graphical data visualization
    • sympy - symbolic computing
    • scipy - lots of generic scientific stuff (contains numpy, sympy, matplotlib, etc)
    • sklearn (SciKit-Learn) - machine learning
    • nltk (Natural Language Toolkit) - text and language processing
    • cv2 (OpenCV) - computer vision
    • PIL (Pillow) - computer vision
    • requests http internet requests
    • wxPython graphics
    • pygame - 2D games
    • nose - testing
