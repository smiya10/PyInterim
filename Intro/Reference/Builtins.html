Builtin Functions List

  • Casting
    • str() casts to a string
    • int() casts to an int
    • float() casts to a float
    • complex() casts to a complex number
    • chr() converts an integer to the Unicode character equivalent
    • ord() converts a Unicode character to its integer value
  • Math
    • abs() absolute value
    • min() minimum of multiple numbers or of a collection of numbers
    • max() maximum of multiple numbers or of a collection of numbers
    • round() rounds value to integer or specific digit
  • Looping
    • range() creates iterator from one value to another by a specific step
    • iter() creates iterator from object (most commonly collection)
    • reversed() creates reverse iterator from object (most commonly collection)
    • zip() allows looping over multiple variables at once
  • Collections
    • list() creates a list from another collection
    • tuple() creates a tuple (we'll discuss it later) from another collection
    • map() creates a map (we'll discuss it later) from another collection using a function
    • len() gets length of collection
    • sum() gets sum of collection number values
    • max() gets maximum of collection number values
    • min() gets minimum of collection number values
    • sorted() creates sorted list from a collection
      • all() true if all values in collection are true
      • any() true if any values in collection are true
  • Input/Output
    • input() reads user input as a string
    • print() displays to the user
    • open() opens a file to read or write
  • Helpful Functions in Interactive Mode
    • help() gives the help information about an object or its type
    • dir() gives a list of an object's attributes
