Using functions

Like in math, functions use variables to calculate values. For example, \( f(x) = 4x^2 \) is a function of x. For the comparable function 4 * x ** 2 in Python (or other programming languages), f is the function x is a parameter and the value of f(x) is the return value. In a purely functional paradigm, functions act just as in math (they accept parameters and return a value based only on the parameters)1. For the example above, the funtion could be called by the following return_value = f(5). Note: "return" is a reserved word in Python. After the above statement, the variable return_variable would contain the return value of the function f with 5 as a parameter, or 100.
