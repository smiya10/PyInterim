Writing Scripts

What is a script?

So far, you hvae written everything interactively (i.e. you type each line and see immediate results). Writing scripts allows you to save and rerun your code. A script is just the code saved as a file instead of written interactively. It's as simple as that.

Advantages

  • Can be reused
  • Runs the exact same code each time it is run

Disadvantages

  • You cannot immediately see results
  • You do not have a second chance to rerun different versions of faulty lines of code

How to run scripts

So, you've written your code and want to run it as a script. In the online interpreter, just insert it into the left panel and click run. If you have Python installed and in your path variable (Windows), name the file with .py extension and run python [filename]. Writing scripts is more like what you would do in any producion setting, so you will use this method for any future work.

Advanced Math Functions

This section is not supported by an example script because these math functions are written the same way as the ones from the previous section.

  • Exponentiation **
    • Unlike many other languages, Python provides a simple functionality for exponentiation.
    • For variables a and b \(a^b\) is simply a ** b
  • Truncating Devision //
    • Originally designed to deal with integer result when dividing two integers
    • For example, using "integer division," 9 / 4 is 2 because the 0.25 of result 2.25 is truncated to become an integer.
    • In Python, this is represented by // and can apply to both integers and flaots
    • For floats, the answer is truncated to the highest multiple of the divisor without going over.
      • Ex. 9.3 // 3.5 is 2.0 because \( 3.5\times 2 = 7\) which is less than \(9.3\) and \(3.5\times 3 = 10.5\) which is greater than \( 9.3 \).
      • Note that the result is a float (signified by the .0) because at least one operand was a float
  • Modulo Operator
    • Also known as modulus
    • Gets remainder after truncating division
    • Symbolized by % symbol (nothing to do with percents)
      • Ex. 9 % 4 is 1 because \( 9 \div 4 = 2 \) with remainder \( 1 \).
    • The same applies for floats but is not always a whole number
      • Ex. 10.8 % 3.1 is 1.5 because 10.8 // 3.1 is 3.0 and \( 10.8 - 3.1 \times 3.0 = 1.5 \).
  • A Note on Floating Point Precision
    • The above example brings up another important topic: floating-point precision.
    • Try the example from above: 10.8 % 3.1.
      • The result is 1.5000000000000004, but it is expected to be 1.5 exactly.
    • Floating-point numbers are stored using a finite number of digits and the power of 10, similar to scientific notation. Occasionally floating-point math becomes slightly off due to rounding error.

String Manipulation

First Question: What is a string?

Strings are collections of characters. Collections are a later topic, but they are just ways of storing multiple items in a single variable. Note that because strings are collections, many properties of lists will also apply to other collections.

Casting

Casting will be shown in the example, but before viewing the example, it is important to know what casting is and why it is used. Casting is the conversion of one data type to another. (i.e. converting a string to an integer). Casting uses built-in functions.
