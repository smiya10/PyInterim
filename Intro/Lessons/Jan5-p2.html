Writing Scripts

What is a script?

So far, you hvae written everything interactively (i.e. you type each line and see immediate results). Writing scripts allows you to save and rerun your code. A script is just the code saved as a file instead of written interactively. It's as simple as that.

Advantages

  • Can be reused
  • Runs the exact same code each time it is run

Disadvantages

  • You cannot immediately see results
  • You do not have a second chance to rerun different versions of faulty lines of code

How to run scripts

So, you've written your code and want to run it as a script. In the online interpreter, just insert it into the left panel and click run. If you have Python installed and in your path variable (Windows), name the file with .py extension and run python [filename]. Writing scripts is more like what you would do in any producion setting, so you will use this method for any future work.
