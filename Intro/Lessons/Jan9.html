Intro to Collections

This section is also contained in an example script. The only thing left to explain is why collections are useful.

Why Collections

Collections are variables which contain multiple pieces of data. Collections make it easier to store multiple related values without cluttering with variables, but more importantly, they are used to contain datasets of variable length. In Python specifically, collections help with looping and functions with multiple return values.

Specifics of Each Collection

  • Tuples
    • Immutable (cannot be changed)
    • Faster than lists
    • Write-protected
  • Strings
    • Contain only characters
    • Immutable
  • Lists
    • Length and values can be changed
    • Not as fast as tuples (but necessary for changing values)
  • Dictionaries
    • Map from one value to another
      • i.e. {"three": 3, "four": 4} means "three" is mapped to 3 and "four" is mapped to 4
    • Fairly slow
