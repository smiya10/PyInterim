A Little About Python

Although it will be covered in more detail, this section briefly describes the philosophy of programming. Computer programming has changed drastically in a short time. Programming languages have been developed to make code easier to develop and less hardware-dependent. Today, there are a wide variety of programming languages to fulfill various needs. This course uses Python because it is easy to learn, but it is necessary to understand the differences between Python and other languages. First, Python is an interpreted language. Unlike compiled languages, the code is not converted to machine code until it is needed. As the a Python program runs, it is interpreted into usable code (because computers only understand binary machine code). This approach is slower than compiled languages, but it allows easier to development in many ways.

Another classification of programming languages divides them into paradigms. A paradigm is simply the approach of completing tasks. The two most common are object-oriented and functional. Other paradigms such as rules-based exist, but are less common. Python is multi-paradigm meaning that it supports both functional and object-oriented paradigms. In this course, you will learn what all this means and how to use the functional paradigm of Python. Python as an object oriented language is not covered in this class, only in the advanced class.
