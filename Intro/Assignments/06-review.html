Mid-Class Review Assignment

This assignment is coding review of everything we've discussed so far. Anything since the beginning is fair game. Make sure to comment your code. Run your code after you write it to test and debug. Good luck.

  1. Write a function which accepts three numbers, the length, width, and height of a rectangular prism then returns the volume and surface area of the prism. (\( V = l \times w \times h\) and \( SA = 2 \times l \times w + 2 \times l \times h + 2 \times w \times h\)). Test the function first with literal values, then allow the user to input the three values (probably floats) and view the volume and surface area. Make sure to tell the user which output is which.
  2. Write a function which accepts 3 values. Using Boolean logic (not loops) calculate and return the maximum of the three values. Test the function with literal values, then provide implementation for user input.
  3. Write a program which accepts two numbers and an operator separated by spaces from the user and then calculates the value requested. (i.e. if the user typed "3 + 9" it would output 12). This would be most appropriate implemented as a function, but you don't have to do that for this problem. To start, only implement the four basic operators +, -, *, and /, but you can implement more if the first works and you feel ambitious. For separating the string on spaces, check out the string function split(). Do not worry about the errors that arise from invalid user input - these can only be resolved with stuff we haven't learned yet.
  4. Write then call a function which accepts a set number of arguments of various types then puts them in a mad lib. Return the mad lib as a single string. You have several choices in putting the items together into a single string (formatting specifiers, format() function, concatenation). In your comments, you can just say the data type since the purpose of the variable is a secret in this case. Ask the user for those variables one-by-one.
