Conditionals Assignment

This assignment attempts to test conditionals and control flow without much dependency on other lessons (although some is anavoidable). Remember to comment your code. Good luck.

Problem Number Problem Code Solution

Ask the user for a single word, then do the following:
  • Introduce yourself if the word is "hello"
  • Say goodbye if "bye"
  • Say "stop, drop and roll" if "fire"
  • Say "coffee" if the word is "sleep"








Ask the user to input the number of sides on two dice. (So two integers.) Use the random.randint() function to simulate the rolling of each die. Print which die won the roll.










Pick a random number between 0 and 10 (again with random.randint() ). Ask the user for a number and tell them if they guessed it correctly. If they were wrong, tell them how far they were from the number.










Ask the user for two number values and do the following:
  • Say "yes" if either is 42
  • Print the sum of the numbers if they are both in the range [0, 10) (meaning greater than or equal to 0 and less than 10)
  • Print the first divided by the second if the second is not zero
  • Print the product if nothing else works
