Getting Input

Let's take a breather from the heavy stuff and do something simple but crucial. In the first lesson, you learned how to print things to the console. In this lesson, we will do the opposite--input.
So let's take a look. Go ahead and type the following code in the interpreter:

input("Give me some input: ")

Note: It is essential that you are running Python 3 for this lesson. The input function behaves differently in Python 2.

So, you've probably figured out how input works. The string you pass input is the prompt and the return value of input is, quite reasonably, the input as a string. Makes sense, right? Now, we can move on to our first in-class project: CopyCat!
