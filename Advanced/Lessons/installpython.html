Python Installation and Setup

On Linux

Open up your terminal (Go to your menu and type terminal). Now, type python --version. If you see 'Python 3.something' you're good. If you see 'Python 2.something', try python3 --version. If, any point, you get command not found, you need to install python.

Installing Python

Go here and download Gzipped source tarball. Once it downloads, click the download. You should see the file and an extract button. Press Extract and note where the files are. Then, open up your terminal and follow the instructions in the readme.

On Windows

Unless you've installed Python before, you don't have Python. This is the website to install the correct version of Python. You want to install the Windows executable installer. If you have a 64-bit machine, you want the x86 version. If you have a 32-bit machine, or just don't know, install the regular x86. Click the download as you normally would and follow the instructions.
